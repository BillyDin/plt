## Livraison 2.1

## Get Started

Clone the repository
```sh 
$ git clone <projectUrl>
```

## Build

```sh
$ cd plt
$ mkdir build
$ cd build
$ cmake .. && make
```

## Print

```sh
$ ./bin/client hello
```

## Test

```sh
$ cd ./plt/build
$ make code-coverage
```

## Print a rendering

```sh
$ ./bin/client render
```
